# KiKoGame - Web Version

Веб-версия игры KiKoGame, полностью переписанная на JavaScript/HTML5 Canvas.

## Особенности

- ✅ Полностью работает в браузере
- ✅ Сохранена вся концепция и дизайн оригинальной игры
- ✅ Все функции работают идентично Python версии
- ✅ Готова к деплою на Vercel, GitHub Pages или любой другой хостинг

## Запуск

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   python -m http.server 8000
   # или
   npx serve
   ```
3. Откройте http://localhost:8000/web/

## Деплой на Vercel

1. Убедитесь, что все файлы в папке `web/` загружены
2. В Vercel создайте новый проект
3. Укажите корневую папку как `web/`
4. Деплой!

## Структура файлов

- `index.html` - главный HTML файл
- `style.css` - стили
- `config.js` - конфигурация игры
- `main.js` - главный игровой цикл
- `spaceship.js` - класс ракеты
- `enemy.js` - класс врагов (астероиды)
- `department.js` - класс департаментов
- `key.js` - класс ключей здоровья
- `planet.js` - класс финальной планеты
- `background.js` - класс фона
- `scores.js` - система очков и UI
- `quiz.js` - система тестов
- `start_screen.js` - меню и правила
- `events.js` - система событий и таймеров
- `sound.js` - управление звуками
- `departments_data.js` - данные о департаментах и вопросах
- `utils.js` - вспомогательные функции

## Примечания

- Все изображения и звуки должны быть в папке `PICS/` на уровень выше `web/`
- Игра использует Canvas API для отрисовки
- Звуки работают через HTML5 Audio API
